项目介绍

一、项目背景

自2018年起，中美之间爆发了现代历史上规模最大、影响最深远的贸易冲突之一，涉及数千亿美元的商品及核心产业。作为世界两大经济体，这场贸易战不仅重塑了全球供应链格局，也深刻影响了世界经济走势与多边贸易秩序。

二、平台宗旨

本平台以中美贸易战为核心主题，通过动态图表、数据可视化与结构化信息，为用户提供一站式观察窗口，帮助公众理解政策演变、掌握宏观趋势、分析经济影响。

三、核心功能

  • 中国进出口总值可视化：呈现自2018年以来中国对美贸易总值变化，分季度/年度动态趋势。
  • 中美双边贸易变化折线图：反映贸易战前后进出口比例与品类结构的重大调整。
  • 网络热议检测：直观可见网络热议话题并从中提炼高频词语形成云图。

四、技术实现

平台采用HTML + CSS + JavaScript构建前端架构，搭配ECharts图表库实现高质量数据可视化，兼容桌面与移动端浏览体验。同时，为后期集成 Python 后端分析与自动更新数据预留拓展接口。

五、设计理念

网站致力于将复杂的贸易数据转化为直观易懂的图表，通过折线、柱状和饼图等形式，让用户在最短时间内捕捉中美进出口与双边贸易变化的核心趋势。我们采用 React 与模块化组件化架构，配合按需加载和响应式布局，保证页面在各种设备和网络环境下都能流畅运行。简洁而富有科技感的界面配合细腻动画交互，不仅提升了数据探索的体验，也便于后续快速扩展新功能。所有代码开源共享，以期汇聚更多智慧，共同打造一个可持续演进的贸易数据可视化生态。

六、适用人群

本项目面向高校学生、研究者、政策分析师及关注国际局势的公众读者，旨在提供多维度、多形式的数据洞察与结构化信息参考。
